# AFIS.IQ.K

Biblioteca de enlace dinámico (.dll o .so) que contiene el algoritmo NFIQ de NBIS (<a href="https://www.nist.gov/services-resources/software/nist-biometric-image-software-nbis">NIST Biometric Image Software</a>) para el cáculo de la calidad de las imágenes que contienen impresiones dactilares. 

Compilación para Linux (CodeBlocks) y Windows (VS).

nfiq.h

Probada desde .NET con AFIS.IQ

Probado su funcionamiento en entornos multi hilos (thread safe)

The non-export controlled NBIS software includes five major packages: (PCASYS, MINDTCT, NFIQ, AN2K7, and IMGTOOLS).
